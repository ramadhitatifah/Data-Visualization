# Visualisasi Garis Kemiskinan Per Kapita Jawa Barat

Dashboard ini dibuat menggunakan Looker Studio untuk menampilkan data garis kemiskinan per kapita bulanan berdasarkan kabupaten/kota di Jawa Barat.

## Link Dashboard

https://datastudio.google.com/u/0/reporting/306adf22-d2e9-43b1-8011-71daa9a025a7/page/ajCkD/edit

## Isi Dashboard

- Peta persebaran garis kemiskinan per kapita
- Donut chart berdasarkan tahun
- Donut chart berdasarkan kota/kabupaten
- Bar chart perbandingan tiap kota/kabupaten
- Tabel nilai per kapita tahun 2018, 2019, dan 2020

## Tools

- Looker Studio
- Google Sheets / sumber data terkait
- GitHub untuk dokumentasi dan versioning
